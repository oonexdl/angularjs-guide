# angularjs-guide

## start

```shell
docker pull node
docker run --rm -v $(pwd)/src:/src -w /src node npm install

docker pull nginx
docker built -t my-nginx .
docker run -d -p 80:80 --name my-nginx my-nginx
```

- demo: http://localhost
